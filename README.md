# AI SEO Automation Toolkit

This project provides open-source automation workflows for SEO and content systems.

The goal is to help developers, marketers, and small teams automate common SEO tasks.

## Features

- Keyword clustering automation
- AI blog generation workflow
- Technical SEO audit analysis
- Internal linking automation
- Content outline generation

## Tools Used

- WordPress
- Google Sheets
- n8n automation
- AI models like Claude

## Example Use Cases

1. Automatically generate blog articles from keywords.
2. Analyze website crawl reports.
3. Create internal linking suggestions.
4. Generate SEO content outlines.

## Why This Project

Many developers and marketers want to automate SEO tasks but lack technical tools.  
This project provides simple automation workflows that anyone can use.
